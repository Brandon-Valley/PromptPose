World-space joint positions

World-space joint positions are the actual 3D locations of each body part after all the rotations are applied.

Example: instead of only saying the left shoulder rotated 40 degrees, it can also say the left hand is at x -1.25, y 1.80, z 0.10.

Leave this on when you want the LLM to understand or preserve the visible pose shape. Turn it off when you want a shorter prompt, especially for long animations.

Downloadable JSON file output

This file-name option appears automatically when the requested animation is at least 50 frames.

Long animation JSON can be too large for normal copy/paste in some LLM apps. When this appears, the generated prompt tells the LLM to provide its answer as a downloadable .json file using this file name.

You can then use Import .json file in Backup & Restore to load that file back into this pose tool.
